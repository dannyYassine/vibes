# CostCoop - Features

## Requester Features

### Core (MVP)
- [ ] Browse Costco food court menu by store location
- [ ] Select a Costco location (manual store selector)
- [ ] Add items to cart with quantity and customization notes
- [ ] Place an order and pay in-app
- [ ] View real-time order status (polling-based)
- [ ] Push notifications for order updates (accepted, picked up, delivered)
- [ ] View order history

### Enhanced (Post-MVP)
- [ ] Rate runners after delivery (1-5 stars + optional comment)
- [ ] View runner ratings before order is accepted
- [ ] Save favorite orders for quick re-ordering
- [ ] Schedule orders for a future time
- [ ] Estimated wait time based on runner availability and store distance
- [ ] Order tracking via polling

### Social (Post-MVP)
- [ ] Share orders with friends via link/invite
- [ ] Group orders — multiple requesters combine into one pickup
- [ ] Referral system — invite friends, earn credits
- [ ] Social feed of friends' recent orders (opt-in)

---

## Runner Features

### Core (MVP)
- [ ] Toggle availability on/off
- [ ] View available orders near selected Costco location
- [ ] Accept an order (broadcast, first-come-first-served)
- [ ] View order details (items, requester notes, delivery address)
- [ ] Mark order as: purchased → picked up → delivered
- [ ] Receive payment (food reimbursement + service fee + tip)

### Enhanced (Post-MVP)
- [ ] Earnings dashboard (daily, weekly, monthly totals)
- [ ] Delivery stats and history
- [ ] Batch multiple orders from the same store for efficiency
- [ ] Route optimization for multi-order deliveries

### Gamification (Post-MVP)
- [ ] Achievement badges (first delivery, 10 deliveries, 100 deliveries, etc.)
- [ ] Leaderboards (top runners by store, by region)
- [ ] Streak bonuses (consecutive days with deliveries)
- [ ] Runner levels/tiers with perks
